# Student Internship Registration System

A web application to manage student internship registrations with OTP verification.

## Features
- Student registration form
- Email OTP verification
- Admin dashboard
- PostgreSQL database storage
- Document upload support

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- HTML / CSS
- Nodemailer
- Multer

## Installation

1. Clone the repository
git clone https://github.com/PavanKumarPadala6/student-internship-registration-system

2. Install dependencies
npm install

3. Run the server
node server.js
